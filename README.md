# CargoScope

[CargoScope](https://cargoscope.io/) is an ocean container tracking platform for freight teams, providing shipment visibility across 100+ ocean carriers.

Track container journeys, monitor ETA and milestone changes, follow rail and inland legs, and share live tracking pages with customers and teams.

## Official website

Visit **[cargoscope.io](https://cargoscope.io/)** to learn more or start tracking.

This repository hosts a simple public information page for CargoScope.
